# Event ui

Its backend is [events](https://github.com/Blackbaud-DiHuynh/events) 

### Prerequisite
* make sure `events` is in `service-binding.yml`
* bluemoon-local-router > gw startNIP

### To serve project: 
```bash
nvm use
npm install
skyux serve --launch local
```
